# 📩 Spam Message Detector

## 👩‍💻 Project Overview  
This project is a simple machine learning application that detects whether a message is **Spam** or **Not Spam (Ham)**.

It was developed as part of an AI internship to understand how real-world text data can be analyzed and classified using basic NLP (Natural Language Processing) techniques.

---

## 🤖 How It Works  
The system follows a simple process:

- Takes a message as input  
- Converts the text into numerical format using **CountVectorizer**  
- Uses a **Naive Bayes classifier** to predict whether the message is spam or not  

---

## ✨ Features  
- Classifies messages as **Spam** or **Ham**  
- Accepts user input in real time  
- Uses basic NLP techniques  
- Beginner-friendly implementation  

---

## 🛠 Technologies Used  
- Python  
- Pandas  
- Scikit-learn
